# 🛒 E-Commerce Website Using HTML, CSS & Bootstrap

## 📌 Project Title
**E-Commerce Website with Bootstrap CSS Responsiveness**



## 📖 Project Description
This is a simple and responsive E-Commerce Website developed using HTML, CSS, Bootstrap, and JavaScript.  
The project demonstrates basic front-end development concepts including user registration, login authentication, product catalog browsing, and cart functionality.

The website is designed with responsive layout using Bootstrap to ensure compatibility across desktop, tablet, and mobile devices.



## 🚀 Features
- ✅ User Registration Page  
- ✅ Login Page  
- ✅ Product Catalog Page  
- ✅ Shopping Cart Page  
- ✅ Logout Functionality  
- ✅ Bootstrap Responsive Design  
- ✅ Clean and Structured UI  



## 🛠 Technologies Used
- HTML5  
- CSS3  
- Bootstrap 5
